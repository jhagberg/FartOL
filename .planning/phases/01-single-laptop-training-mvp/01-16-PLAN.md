XML shape (RESEARCH §Pattern 7 + Pitfall 5 conservative subset). Top-level:
```
<?xml version="1.0" encoding="UTF-8"?>
<ResultList xmlns="http://www.orienteering.org/datastandard/3.0" iofVersion="3.0" createTime="<ISO>" creator="FartOL v0.1" status="Complete|Snapshot">
  <Event>
    <Name>{competition.name}</Name>
    <StartTime><Date>{competition.date}</Date></StartTime>
  </Event>
  <!-- Per class — minOccurs=0; empty competitions emit zero ClassResult children -->
  <ClassResult>
    <Class><Name>{class.name}</Name></Class>
    <!-- Per competitor -->
    <PersonResult>
      <Person>
        <Name><Family>{family}</Family><Given>{given}</Given></Name>
      </Person>
      <Organisation type="Club"><Name>{club}</Name></Organisation>   <!-- only if club non-null -->
      <Result>
        <StartTime>{ISO start}</StartTime>                            <!-- only if known -->
        <FinishTime>{ISO finish}</FinishTime>                          <!-- only if known -->
        <Time>{elapsed seconds}</Time>                                 <!-- integer seconds -->
        <Position>{place}</Position>                                   <!-- only if status=OK and place set -->
        <Status>{OK|MissingPunch|DidNotFinish}</Status>
      </Result>
    </PersonResult>
  </ClassResult>
</ResultList>
```

**@status attribute mapping (LOCKED — W-4):**
- ResultListStatus enum values per IOF 3.0 XSD: `Complete | Delta | Snapshot | Refused`.
- `status: 'Final'` → emit `@status="Complete"` (the export represents the closed competition).
- `status: 'Provisional'` → emit `@status="Snapshot"` (an in-progress point-in-time view; the IOF-canonical value for "results-so-far").
- We always emit @status (no implicit-default branch); this simplifies parser interop. Verify the enum against apps/edge/iof-3.0/IOF.xsd (plan 05 Wave 0) — if either value is absent in the bundled XSD version, the executor adjusts: 'Complete' may appear as 'Final' in some 3.0 dialect; in that case use whichever the XSD accepts and document in summary.

**Empty competition (LOCKED — W-5):**
- Per the IOF 3.0 XSD: `<xs:element name="ClassResult" minOccurs="0" maxOccurs="unbounded">`. Empty competition exports are VALID XML.
- The export endpoint returns 200 with a valid XML body containing zero ClassResult children when there are no non-PEND results to emit. The validator must accept this (test 5 below asserts the round-trip).

Status map per IOF spec (per-competitor status, not @status):
- 'OK' → 'OK'
- 'MP' → 'MissingPunch'
- 'DNF' → 'DidNotFinish'
- 'PEND' → omitted (don't emit PersonResult for pending starters; they get re-exported when they finish).

PEND competitors are EXCLUDED from the export entirely. ResultList semantics target finished events; pending starters get re-exported when they finish. Phase 2 operator-controlled "include not-started" flag can land later. The provisional toggle does NOT change this; provisional means "results are final-for-now, may be amended."

Name splitting: competitor.name is one string "Given Family"; split on last space (Swedish naming convention: last token is family). If only one token, Family=token, Given=''. The XSD requires both `<Family>` AND `<Given>` per the v3.0 schema; emit empty `<Given/>` if needed.

Time format: IOF XML 3.0 ResultList `<Time>` is decimal seconds (per the XSD's xsd:decimal). E.g., 12:30 elapsed → `<Time>750</Time>`. `<StartTime>` and `<FinishTime>` are xsd:dateTime ISO 8601 (combine competition.date with the wall-clock derived from competitor's start_time_ms relative to local timezone — for Phase 1, derive from event_time_ms as a Date.toISOString()).

Export endpoint shapes:
- GET /api/competitions/:id/export/preview?status=Final|Provisional → 200 { valid: true, summary: { class_count, person_result_count, status } } OR { valid: false, errors: XsdError[] }. For empty competitions: valid=true, class_count=0, person_result_count=0.
- GET /api/competitions/:id/export?format=iof30&amp;status=Final|Provisional → 200 with Content-Type: application/xml + Content-Disposition: attachment; filename="<slug>-resultlist.xml" — the XML body (which may have zero ClassResult children). Validation runs first; on fail, return 400 with errors[] (NOT a partial file).

    import { XMLBuilder } from 'fast-xml-parser';
    import { validateXml } from './validate.ts';
    import type { CompetitionState, CompetitorView, ResultView } from '../projection/types.ts';

    export interface ExportInput { competition: CompetitionDTO; classes: ClassDTO[]; courses: CourseDTO[]; state: CompetitionState; status?: 'Final' | 'Provisional'; creator?: string; }
    export interface ExportSummary { class_count: number; person_result_count: number; status: 'Final' | 'Provisional'; }
    export interface BuildResult { xml: string; summary: ExportSummary; }
    export interface ValidatedBuildResult { valid: boolean; errors?: import('./validate.ts').XsdError[]; build?: BuildResult; }

    function splitName(full: string): { family: string; given: string } { const trimmed = full.trim(); const idx = trimmed.lastIndexOf(' '); if (idx < 0) return { family: trimmed, given: '' }; return { family: trimmed.slice(idx + 1), given: trimmed.slice(0, idx) }; }
    function statusForXml(s: CompetitorView['status']): string | null { return ({ OK: 'OK', MP: 'MissingPunch', DNF: 'DidNotFinish', PEND: null } as const)[s]; }
    function resultListStatusFor(input: 'Final' | 'Provisional'): 'Complete' | 'Snapshot' {
      // W-4 LOCKED: ResultListStatus enum from IOF.xsd is Complete | Delta | Snapshot | Refused.
      // Final → Complete; Provisional → Snapshot.
      return input === 'Final' ? 'Complete' : 'Snapshot';
    }
    function buildPersonResult(v: CompetitorView): unknown | null {
      const xmlStatus = statusForXml(v.status); if (!xmlStatus) return null;  // PEND skipped
      const { family, given } = splitName(v.name);
      const pr: any = { Person: { Name: { Family: family, Given: given || { '#text': '' } } } };
      if (v.club) pr.Organisation = { '@_type': 'Club', Name: v.club };
      const res: any = { Status: xmlStatus };
      if (v.elapsed_time_ms !== null) res.Time = Math.floor(v.elapsed_time_ms / 1000);
      // Position injected by caller (needs per-class ResultView).
      pr.Result = res;
      return pr;
    }

    export function buildResultListXml(input: ExportInput): BuildResult {
      const status = input.status ?? 'Final';
      const creator = input.creator ?? 'FartOL v0.1';
      const xmlStatusAttr = resultListStatusFor(status);
      const classResults: unknown[] = [];
      let personResultCount = 0;
      for (const cls of input.classes) {
        const rows = input.state.results_by_class.get(cls.id) ?? [];
        const personResults: any[] = [];
        for (const row of rows) {
          const view = input.state.competitors.get(row.competitor_id);
          if (!view) continue;
          const pr = buildPersonResult(view) as any;
          if (!pr) continue;
          if (view.status === 'OK' &amp;&amp; row.place !== null) pr.Result.Position = row.place;
          personResults.push(pr);
          personResultCount++;
        }
        if (personResults.length === 0) continue;
        classResults.push({ Class: { Name: cls.name }, PersonResult: personResults });
      }
      // W-5 LOCKED: empty competition (classResults.length === 0) emits a valid empty ResultList.
      // Do NOT bail with an error; ResultList > ClassResult is minOccurs=0.
      const resultListNode: any = {
        '@_xmlns': 'http://www.orienteering.org/datastandard/3.0',
        '@_iofVersion': '3.0',
        '@_createTime': new Date().toISOString(),
        '@_creator': creator,
        '@_status': xmlStatusAttr,                           // W-4: always emit
        Event: { Name: input.competition.name, StartTime: { Date: input.competition.date } },
      };
      if (classResults.length > 0) resultListNode.ClassResult = classResults;
      const tree = {
        '?xml': { '@_version': '1.0', '@_encoding': 'UTF-8' },
        ResultList: resultListNode,
      };
      const builder = new XMLBuilder({ ignoreAttributes: false, attributeNamePrefix: '@_', format: true, indentBy: '  ', suppressEmptyNode: false });
      const xml = builder.build(tree);
      return { xml, summary: { class_count: classResults.length, person_result_count: personResultCount, status } };
    }

    export function validateAndBuild(input: ExportInput): ValidatedBuildResult {
      const built = buildResultListXml(input);
      const v = validateXml(built.xml);
      if (!v.valid) return { valid: false, errors: v.errors };
      return { valid: true, build: built };
    }

    import { eq } from 'drizzle-orm';
    import { competitions, classes as classesTable } from '../db/schema.ts';

    export default async function exportRoutes(app: FastifyInstance) {
      app.get('/api/competitions/:id/export/preview', async (req, reply) => {
        const { id } = req.params as { id: string };
        const status = (req.query as any).status === 'Provisional' ? 'Provisional' : 'Final';
        const input = loadCompetitionInputs(app.fartolDb, id);
        if (!input) return reply.code(404).send({ error: 'competition_not_found' });
        const comp = app.fartolDb.db.select().from(competitions).where(eq(competitions.id, id)).get()!;
        const classes = app.fartolDb.db.select().from(classesTable).where(eq(classesTable.competitionId, id)).all();
        const state = app.projectionStore.recomputeNow(id)!;
        const result = validateAndBuild({ competition: { ...comp, auto_print: !!comp.autoPrint, receipt_template: comp.receiptTemplate, created_at_ms: comp.createdAtMs } as any, classes: classes.map(c => ({ ...c, competition_id: c.competitionId, short_name: c.shortName })) as any, courses: input.courses as any, state, status });
        if (!result.valid) return reply.code(200).send({ valid: false, errors: result.errors });
        return reply.send({ valid: true, summary: result.build!.summary });
      });

      app.get('/api/competitions/:id/export', async (req, reply) => {
        const { id } = req.params as { id: string };
        const format = (req.query as any).format;
        const status = (req.query as any).status === 'Provisional' ? 'Provisional' : 'Final';
        if (format !== 'iof30') return reply.code(400).send({ error: 'unsupported_format' });
        const input = loadCompetitionInputs(app.fartolDb, id);
        if (!input) return reply.code(404).send({ error: 'competition_not_found' });
        const comp = app.fartolDb.db.select().from(competitions).where(eq(competitions.id, id)).get()!;
        const classes = app.fartolDb.db.select().from(classesTable).where(eq(classesTable.competitionId, id)).all();
        const state = app.projectionStore.recomputeNow(id)!;
        const result = validateAndBuild({ competition: { ...comp, auto_print: !!comp.autoPrint, receipt_template: comp.receiptTemplate, created_at_ms: comp.createdAtMs } as any, classes: classes.map(c => ({ ...c, competition_id: c.competitionId, short_name: c.shortName })) as any, courses: input.courses as any, state, status });
        if (!result.valid) return reply.code(400).send({ error: 'xsd_invalid', errors: result.errors });
        const slug = comp.name.toLowerCase().replace(/[^a-z0-9]+/g, '-').replace(/^-+|-+$/g, '') || 'competition';
        reply.header('Content-Type', 'application/xml; charset=utf-8');
        reply.header('Content-Disposition', `attachment; filename="${slug}-resultlist.xml"`);
        return reply.send(result.build!.xml);
      });
    }
