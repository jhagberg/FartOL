## Implementation Details

### Task 1 — DB + Auth functions
- `event_codes` table: id, competition_id (FK→competitions cascade), code, expires_at_ms, revoked_at_ms (nullable), created_at_ms
- Two indexes: `idx_event_codes_comp_code` (lookup path) and `idx_event_codes_comp_active` (expiry queries)
- Migration 0010 uses `-->statement-breakpoint` delimiter (no comments in SQL — better-sqlite3 multi-statement constraint)
- 35-word locked Swedish O-feature wordlist (åsen, sänkan, höjden, …, terrassen) — never extended without migration
- `generateCode()`: `crypto.randomInt(0,35)` word + `crypto.randomInt(100,1000)` number
- `validateCode()`: regex pre-check `^[a-zåäö]+-[1-9][0-9]{2}$` then DB query with AND(eq cid, eq code, gt expiresAtMs, isNull revokedAtMs)
- `signCookie()` / `verifyCookie()`: HMAC-SHA256, base64url(JSON payload).base64url(sig), `timingSafeEqual` constant-time comparison

### Task 2 — Routes + preHandler gate
- POST/GET/revoke `/api/competitions/:id/event-codes` — localhost gate via `socket.remoteAddress` (never XFF per T-02.1-27)
- `maskCode()`: first 3 chars + `****` + last 2 (e.g. `sänkan-127` → `sän****27`)
- `getOrCreateSigningSecret()`: reads or auto-generates+persists 32-byte hex secret in config table
- POST `/access`: revoked → expired → validateCode → recordAttempt → signCookie → `fartola_event_code` HttpOnly SameSite=Lax cookie
- In-memory rate limiter: 10 attempts/60s window + exponential backoff (1s→2s→4s… cap 60s) after 5 consecutive failures; cleanup every 5 min
- Blanket `onRequest` hook: gates POST/PATCH/DELETE under `/api/competitions/` for non-localhost; excludes `/event-codes` paths
- Pino redact paths: `body.code`, `*.body.code`, `req.body.code`, `request.body.code`, `body.event_code`, `*.body.event_code`
- ADR-0010: 60+ line document covering entropy, rate-limit math, cookie design, localhost bypass rationale, alternatives

### Task 3 — Web UI
- `AccessView.svelte`: competition picker (auto-selects if only 1), code input (monospace, no autocorrect), POST /access, redirects to /competition/:id/registration on success, localised error messages
- `/access/+page.svelte`: SvelteKit entry point, 3-line wrapper
- `SettingsView.svelte` Hjälpkoder section: generate button → POST + 30-second plaintext reveal (auto-hides), masked list, per-code revoke button, expiry display, revoked opacity
- 22 i18n keys added (11 access.* + 11 settings.helperCodes.*) in sv.json and en.json

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Migration multi-statement error**
- **Found during:** Task 1
- **Issue:** Initial migration SQL used block comments + multiple statements without `-->statement-breakpoint`. better-sqlite3 rejects multi-statement strings.
- **Fix:** Rewrote SQL using `-->statement-breakpoint` delimiter matching existing 0007 migration style. No comments in SQL.
- **Files modified:** `apps/edge/drizzle/0010_event_codes.sql`
- **Commit:** 93bdd69

**2. [Rule 1 - Bug] Test used `handle.db.prepare()` instead of `handle.sqlite.prepare()`**
- **Found during:** Task 1
- **Issue:** Drizzle ORM handle doesn't expose `prepare()`; must use raw better-sqlite3 `.sqlite` property.
- **Fix:** Changed all raw SQL inserts in tests to `handle.sqlite.prepare()`.
- **Commit:** 93bdd69

**3. [Rule 1 - Bug] FK constraint — missing parent competition row in tests**
- **Found during:** Task 1
- **Issue:** Inserting into `event_codes` without a parent `competitions` row triggered SQLITE_CONSTRAINT_FOREIGNKEY.
- **Fix:** Added `insertCompetition(handle, competitionId)` helper using `INSERT OR IGNORE INTO competitions`.
- **Commit:** 93bdd69

**4. [Rule 1 - Bug] Test expiry in the past**
- **Found during:** Task 1
- **Issue:** Competition date `'2026-05-24'` + 24h was already expired before tests ran.
- **Fix:** Changed test competition date to `'2026-12-31'` across all test files.
- **Commit:** 93bdd69, 96596f1

**5. [Rule 1 - Bug] preHandler fired before admin route's own localhost gate**
- **Found during:** Task 2 (green phase, test 7 in event-codes.test.ts)
- **Issue:** Blanket `onRequest` hook returned `event_code_required` for non-localhost POST to `/api/competitions/:id/event-codes` before the route handler could return `localhost_required`.
- **Fix:** Added `if (url.includes('/event-codes')) return;` exclusion to the preHandler; also excluded `/event-codes` paths from the blanket gate.
- **Files modified:** `apps/edge/src/server.ts`
- **Commit:** 5f8f1ce

**6. [Rule 3 - Blocking] `@fastify/cookie` install failed with lefthook hooks path error**

- **Issue:** `pnpm --filter @fartola/edge add @fastify/cookie` failed on the `prepare` lifecycle script because lefthook couldn't find git hooks in the worktree context.
- **Fix:** Used `--ignore-scripts` flag: `pnpm --filter @fartola/edge add @fastify/cookie --ignore-scripts`.
- **Commit:** 5f8f1ce

**7. [Rule 3 - Blocking] 0010_snapshot.json prettier formatting rejected by pre-commit hook**
